The Lab

ReSTful Blog

Simple (browse, publish, edit, delete) blog application leveraging Node.js, Express, MongoDB, and Mongoose. To test drive a new CSS framework, it's also based on Semantic UI vs Bootstrap.

Mostly an academic exercise to practice implementing web services using routes adhering to ReSTful best practices.

Circle Tap

Highly simplified Patatap clone leveraging the free and amazingly awesome Paper.js and Howler.js. Dance party in your browser!

To-Do List

Experimenting with jQuery by building a simple To-Do list. I tend to prefer pure JavaScript, but it's amazing how much you can do with jQuery in so few lines of code! Note to self: needs persistence layer.

Color Guessing Game

Mid-term project from Colt Steele's Full-Stack Web Development Bootcamp. Generates a grid of random colors, and let's users try and guess the RGB values. More practice with HTML, CSS and Javascript.
